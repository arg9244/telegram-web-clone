# Telegram Web Clone

A modern, responsive Telegram Web-inspired UI clone built with HTML, CSS, and JavaScript.

## Features

- **Modern Dark Theme**: Clean, Telegram-inspired dark interface
- **Chat List Sidebar**: With avatars, names, timestamps, and message previews
- **Message View Area**: Clean message bubbles with sent/received styling
- **Interactive Messaging**: Send messages with automatic reply simulation
- **Search Functionality**: Filter chats by name or message content
- **Responsive Design**: Adapts to different screen sizes
- **Smooth Animations**: Subtle transitions and message animations
- **Floating Particles**: Ambient background effects

## Tech Stack

- HTML5 for structure
- CSS3 for styling (variables, flexbox, animations)
- Vanilla JavaScript for interactivity

## Getting Started

Simply open `index.html` in your web browser to view the Telegram Web Clone.

## Project Structure

```
telegram-web-clone/
├── index.html      # Main HTML structure
├── styles.css      # Modern dark theme styling
└── script.js       # Interactive functionality
```
